# CulturePass

CulturePass is a mobile platform designed to connect and celebrate the vibrant Malayalee community. Our mission is to make it easy for you to discover local events, connect with community organizations, and explore the rich cultural landscape around you.

## The Vision

In a world where digital connection is paramount, CulturePass aims to be the go-to digital gateway for the Malayalee community. We want to foster a stronger sense of community by bringing people together, both online and offline. Whether you're looking for a cultural event, a local business, or a community group, CulturePass is your trusted guide.

## Key Features

- **Event Discovery:** Find a comprehensive list of cultural events, from festivals and concerts to workshops and social gatherings.
- **Community Hubs:** Connect with local Malayalee organizations, get updates, and become a member.
- **Business Directory:** Discover and support local Malayalee-owned businesses in your area.
- **Artist Profiles:** A dedicated space for artists to showcase their work and for the community to discover amazing talent.
- **Exclusive Perks:** Unlock special offers and benefits from local businesses and organizations.

## Tech Stack

CulturePass is built with a modern tech stack to ensure a seamless and scalable experience:

- **Frontend:** React Native, Expo
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL with Drizzle ORM
- **Authentication:** Supabase

## Project Structure

The project is organized into the following main directories:

- `/app`: Contains all the screens and navigation logic for the Expo app.
- `/components`: Reusable UI components used throughout the app.
- `/constants`: Global constants like colors and styles.
- `/lib`: Helper functions and libraries, including authentication and data fetching.
- `/server`: The Node.js backend, including API routes, database configuration, and business logic.
- `/shared`: Code that is shared between the frontend and backend, like data schemas.
